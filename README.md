# Superstore Sales Data Analysis

## Project Objective
Analyze retail sales data to understand product performance, regional trends, and customer segments.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Key Insights
- Technology category generates highest sales
- West region performs best
- New York City has highest sales
- Consumer segment contributes most revenue
- Phones are the top selling sub-category
